## Example file

```txt filename="update-page-title.beek" copy
The language is typescript and the frameworks are react.js and next.js.
Just update h1 tag with text content "{{title}}".
```

> Note: The extension for this configuration file needs to be .beek

## Variables binding

Rules can be defined only in files with .beek extension.

These files support variables binding with `{{my_var}}` syntax.
